# 🏢 Internal Communications Platform Comparison Dashboard

A comprehensive, interactive comparison tool for evaluating internal communication platforms designed specifically for organizations with deskless workforces.

## 🚀 Live Demo

**🌐 [View Live Dashboard](https://communications-phi.vercel.app)**

## 📊 Platforms Analyzed

1. **WorkVivo** - Employee Experience Platform
2. **Staffbase** - Employee Communications Platform
3. **Reward Gateway** - Employee Engagement & Recognition
4. **Microsoft Viva Engage** - Communities and Conversations
5. **Simpplr** - Modern Intranet Platform

## ✨ Features

### 📈 Overview Tab
- **Interactive score charts** comparing overall platform performance
- **Comprehensive comparison table** with strengths and limitations
- Visual rankings based on weighted feature scores

### ✅ Feature Matrix Tab
- **Must-Have Features** for deskless workers:
  - Native Mobile App (iOS & Android)
  - System Integration capabilities
  - Permission-Based Targeting
  - Approval Workflows
  - External System Links
  - Video Embedding
  - Social Engagement tools
  - Personal Email Login
  - Surveys & Polls (with analytics & anonymous options)
  - Analytics Dashboard

- **Optional Features**:
  - Rewards Programs
  - Recognition Systems
  - Service Awards
  - Peer-to-Peer Shoutouts
  - Manager Recognition
  - Document Storage

### 💰 Pricing Tab
- **Cost estimates** for 1,800 employees
- **Pricing tier comparisons** across all platforms
- **Value ratings** (score vs cost analysis)
- Annual cost projections

### 🎯 Recommendation Tab
- **Smart recommendation engine** with adjustable priorities
- **Interactive sliders** for:
  - Recognition & Rewards importance
  - Mobile-First Experience needs
  - Budget Consciousness
  - System Integration requirements
- **Real-time ranking updates** based on your priorities
- **Detailed rationale** for top recommendation

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ installed
- npm or yarn package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/dailoclaw/communications.git

# Navigate to project directory
cd communications

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 📈 Scoring Methodology

**Overall Score Calculation:**
- Must-Have Features: 80% weight
- Optional Features: 20% weight
- Individual feature ratings: 0-5 scale
- Feature weights: 1-10 scale based on importance

**Recommendation Engine:**
- Combines overall score with user-defined priorities
- Adjustable weighting for Recognition, Mobile, Budget, Integration
- Real-time recalculation based on slider inputs
